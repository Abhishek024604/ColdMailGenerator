# 🚀 Cold Email Generator

A modern web application that helps users generate **high-quality, personalized cold emails** for outreach, networking, sales, and job applications — instantly.

---

## ✨ Features

* ⚡ Generate cold emails in seconds
* 🎯 Multiple use cases (job outreach, sales, networking, etc.)
* 🧠 Smart tone selection (formal, friendly, persuasive)
* 📋 One-click copy to clipboard
* 🎨 Clean, modern UI with smooth animations
* 🔔 Toast notifications for better UX

---

## 🛠️ Tech Stack

### Frontend

* React (Vite)
* Framer Motion
* Lucide React
* Radix UI

### Backend (optional)

* Node.js + Express
* OpenAI / LLM API (or custom model)

---

## 📦 Installation

### 1. Clone the repository

git clone https://github.com/your-username/cold-email-generator.git
cd cold-email-generator

### 2. Install dependencies

npm install

### 3. Run the development server

npm run dev

---

## 🧠 How It Works

1. User inputs:

   * Target role / company
   * Purpose of email
   * Tone selection

2. System processes input

3. AI generates structured cold email

4. User can copy and use instantly
